# rgb2gray usage guide

The program ./rgb2gray converts an input image to grayscale and saves to disk. The program can be used by either executing the python file or using the bash file.

```$ python3 rgb2gray.py [input_color_image_filename] [output_grayscale_image_filename]```

```$ ./rgb2gray [input_color_image_filename] [output_grayscale_image_filename]```

Usage of the bash file requires execution rights on the file. To enable this execute this command in the directory where the file is stored.

```$ chmod +x rgb2gray.sh```
